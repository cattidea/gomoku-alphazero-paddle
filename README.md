# Gomoku-AlphaZero

based on [gomoku-alphazero](https://github.com/cattidea/gomoku-alphazero)

## Build

```bash
python setup.py build_ext -i
rm mcts.c
rm board.c
```

## Train

```bash
python train.py
```

## Play

```bash
python play.py
```
